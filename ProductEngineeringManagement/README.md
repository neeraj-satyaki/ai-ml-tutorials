# Product + Engineering Management

PM + EM skill stacks. IC → tech lead → manager → director → VP transitions.

## Product skills
Product sense. **Jobs-To-Be-Done** customer discovery. User research. **Product strategy** + North Star metric. Prioritization: **RICE, WSJF, Kano, MoSCoW**. Roadmap + OKRs. **Discovery vs delivery** (Marty Cagan). Product-market fit. Launch management. Post-launch retro. Pricing + packaging. Metric trees. **A/B experimentation**. ProductOps tools (Linear, Jira, Notion, Productboard).

## Engineering leadership
IC ladder (L3-L7+) vs manager ladder (EM → SEM → Director → VP → CTO). Manager's compass (*Trillion-Dollar Coach*). Running 1:1s. Performance reviews + calibration. Giving feedback (*Radical Candor*). Career conversations. Hiring: **structured interviews**, rubrics, scorecards. Onboarding 30/60/90. **Team Topologies** (stream-aligned, platform, enabling, complicated-subsystem). Span of control. Decision frameworks (RAPID, DACI). Meeting hygiene. On-call runbooks.

## Process + engineering ops
Agile / Scrum / Kanban. Scale frameworks: SAFe, LeSS, Spotify model (anti-pattern often). Roadmap planning via **Shape Up** (Basecamp). **DORA** + **SPACE** metrics. Incident management + blameless postmortems. Cost / quality / velocity triangle. Platform teams + DevEx. Remote / hybrid / async-first practices.

## Strategy + business
Blue Ocean, Great Game of Business. BCG matrix, Porter's 5 forces. Business Model Canvas. **Unit economics** (CAC, LTV, contribution margin). Cohort analyses. North Star metric vs vanity. OKR design. Moats + value capture. Build vs buy vs partner. M&A primer. Public company financial literacy (10-K, 10-Q). Portfolio-level management.

## Communication
**Amazon-style writing** — clear memos beat slides. **Six-pager format**. Stakeholder alignment. Executive comms. Radical Candor + Crucial Conversations. Influence without authority. Supportive push. Public speaking fundamentals.
