# Bioinformatics + Computational Biology

Sequence/structure/omics + AI-driven drug discovery + clinical.

## Sequence data
FASTA / FASTQ formats. PHRED quality. Short-read alignment (**BWA**, Bowtie2). Long-read (**minimap2**). Variant calling: **GATK**, **DeepVariant**. RNA-seq: STAR, Salmon, Kallisto. ChIP-seq, ATAC-seq. Hi-C 3D genomics. **Single-cell**: Cell Ranger, **Scanpy**, **Seurat**. Long-read platforms (Nanopore, PacBio HiFi). Population genomics (gnomAD). Metagenomics (Kraken2, Kaiju).

## Structural biology
PDB + mmCIF. MSAs. Homology modeling. **AlphaFold 2/3**, **RoseTTAFold**, **ESMFold / OmegaFold**, **Boltz-1 / Boltz-2** (2024), **Chai-1**. Molecular dynamics (GROMACS, AMBER, OpenMM). Docking (AutoDock Vina, Glide). Free energy (FEP, RBFE). Protein design: **RFdiffusion**, **ProteinMPNN**, **Chroma**.

## Omics — proteomics + metabolomics
Mass spec. **MaxQuant**, **FragPipe** proteomics. Metabolomics (XCMS). Lipidomics. Glycomics. **Spatial omics** (Visium, Xenium, MERFISH). Multi-omics integration (MOFA+).

## Data standards
VCF, BAM/SAM/CRAM, GTF/GFF, BED / BigWig. Parquet / Zarr for genomics scale. AnnData / MuData. **Workflows**: **Nextflow** (+ nf-core), Snakemake, **CWL**, WDL (Cromwell).

## AI for drug discovery
Target ID. **Graph NNs on molecules** (D-MPNN, GIN). SMILES / **SELFIES** string representations. Diffusion docking (**DiffDock**). Equivariant GNNs (EGNN, SE(3)-Transformer). Generative chemistry (REINVENT, ChemFormer). RL for molecule design. Boltz / Chroma / RoseTTAFold-AllAtom. **Protein language models** (ESM2, ESM3, ProtT5). Virtual screening. Retrosynthesis (AiZynthFinder). **Self-driving labs** (lab automation + active learning).

## Clinical / health
**FHIR / HL7** standards. DICOM imaging. Radiogenomics. **MIMIC** clinical NLP. Precision medicine. Polygenic risk scores. Rare disease diagnosis. Pharmacovigilance. Clinical trial analysis (R + Python).

## Tools + frameworks
**Biopython**, **Bioconductor** (R), bcbio-nextflow, **Galaxy**, **Nextflow + nf-core**, Snakemake. Seurat, Scanpy, Cell Ranger, Space Ranger. PyMOL / **ChimeraX**. **DeepChem**, **RDKit**, OpenBabel.
