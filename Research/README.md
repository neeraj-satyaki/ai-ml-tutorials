# Research Skills

Paper reading, experimentation, reproducibility, writing. Useful for PhD, industry research, and self-directed learning at the frontier.

## Skills
- **Problem statement** — motivation, question, why now, contribution.
- **Lit review tools**: Zotero, **Elicit**, Semantic Scholar, Scite.
- **Paper reading — 3-pass** (Srinivasan Keshav): quick scan → understand details → reproduce.
- Critical reading + note-taking (Zettelkasten in Obsidian / Logseq / Roam).
- PDF extract + AI summarization pipelines.
- Writing for **NeurIPS / ICML / CVPR / ICLR / ACL / Nature**. Rebuttal writing. Camera-ready checklists. Ethics + broader impact statements. Open review pros/cons.

## Reproducibility
- **NeurIPS Reproducibility Checklist**. Papers With Code badges.
- Log all random seeds. Pin package versions (lockfiles). Data hash.
- **Docker / Singularity** replay.
- Golden datasets versioned (DVC, LakeFS).
- MLflow / W&B artifact tracking.
- Kaggle-style reproducibility practice.
- ACM Reproducibility Badges.
- Honest reporting: mean + CI across seeds, not single run.
- Failure-to-reproduce log.

## Experimentation
- Design of experiments (factorial). Control variables. **Ablation studies**.
- **Strong baselines** — weaker baselines = invalid conclusions.
- Regression toward mean. Sanity-check models (overfit single batch).
- Statistical significance via bootstrap CI. Multiple hypothesis correction.
- Publication-ready result tables (booktabs).
- LR sweeps, seed stability.

## Open science
- arXiv + preprint norms. OpenReview.
- Open datasets. Permissive model-weight licenses.
- Reproducible code (GitHub release-tagged).
- Data citation (DOI, Crossref).
- CC licenses, code licenses (Apache, MIT, GPL).
- Persistent IDs (ORCID).
- Scholarly communication + altmetrics.
