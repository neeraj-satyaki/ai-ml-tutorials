# Career Ladder + Interview Prep

Consolidated. Ladder signals + prep playbook for every round.

## Engineering ladder (signals per level)
- **L1 / Junior** — executes well-scoped tasks; learning fast; small PRs.
- **L2 / SWE** — ships features end-to-end; owns a module; writes tests.
- **L3 / Senior** — owns a subsystem; mentors; technical tradeoffs; unblocks others.
- **L4 / Staff** — cross-team influence; tech lead + architecture; multi-quarter bets.
- **L5 / Principal** — org-wide tech strategy; represents engineering externally.
- **L6 / Distinguished** — industry-level impact; platform direction.
- **L7 / Fellow** — rare; technical vision for company.
- **IC vs Management** — both paths valid; pick based on energy source.
- **Lateral moves** — company, team, or stack pivots reset expectations.

## DSA prep
- Lists: **NeetCode 150**, Grind 75, Blind 75, LeetCode Top 100.
- Company-specific lists (Leetcode company tags).
- Mock interviews: interviewing.io, Pramp, partner practice.
- Timeboxed practice schedule (2-3 problems/day).
- Common mistakes: no clarification, solving fast without talking, giving up too early.
- Topic frequency by company.
- Speed-run sessions (15 min per easy).

## System design prep
- Classic problems: URL shortener, Twitter feed, YouTube, Dropbox, Uber, Slack, chat, Google Maps, rate limiter, distributed cache, web crawler, payments, notifications.
- Standard framework: clarify → estimate → API → data model → high-level → deep-dive → scale → reliability/security/observability.
- Trade-off matrix (CAP, SQL vs NoSQL, sync vs async).
- SLI/SLO deep dive.
- Distributed systems intuition — CAP, PACELC, Paxos/Raft, consistency models.
- Senior vs junior expectations — senior = tradeoffs + failure modes; junior = correctness.

## Behavioral prep
- **STAR** / PAR framework.
- Company-specific:
  - Amazon: **Leadership Principles** (all 16).
  - Google: "Googleyness".
  - Meta: Move Fast, Focus on Long-Term Impact.
  - Netflix: Keeper Test.
- Build **stories bank** of 10-15 covering: conflict, ownership, failure, mentorship, ambiguity, cross-team, customer obsession, data-driven decisions.
- Negotiation + references.
- "Why X, why now" narrative.

## Specialized rounds
- **ML system design** — feature store, online vs batch, metric design, evaluation.
- **LLM system design** — RAG + eval + latency budget + safety.
- Safety / alignment interview.
- **Data engineering** interviews (pipelines, SQL).
- Security interviews (threat models).
- **Frontend design** (component design, state, rendering).
- Mobile system design.
- Platform team interviews (internal tools).

## Offers + negotiation
- **levels.fyi** for comp benchmarks.
- Equity basics: RSU, ISO, refreshers.
- Sign-on + relocation + equity.
- Competing offers triangle (upside, stability, impact).
- Visa / relocation considerations.
- Internal promo vs external offer path.
- Labor market macro (hiring freezes, RIFs) affects leverage.

## Resume + portfolio

- STAR on resume.
- GitHub profile signal (pinned repos, contribution graph).
- OSS contributions = strong signal.
- Portfolio site templates.
- Technical blog as signal.
- Recommendations.
- LinkedIn hygiene.
