# Competitive Programming

Sharp algorithmic practice. Separate from `CS/Algorithms/` theory — this is about solving under 2-3-hour contest pressure.

## DSA topics
Arrays / two-pointers / sliding window / prefix sum. Monotonic stacks/queues. Hashing. Sorting + order statistics. **Binary search + answer search**. Greedy. Interval problems. Bitmask techniques. **DP** — 1D / 2D / bitmask DP / digit DP / probabilistic DP + optimizations (Knuth, CHT, SMAWK, D&C). Trees: DFS / BFS / Euler tour. LCA (binary lifting + sparse table). RMQ. **Fenwick (BIT)**, **Segment Tree + lazy propagation**, persistent seg tree, merge sort tree. **Sqrt decomposition + Mo's algorithm**. Centroid decomposition. **HLD**. Link-cut / splay. Treap. Graphs: BFS/DFS, Dijkstra / Bellman-Ford / Floyd / 0-1 BFS. MST (Kruskal / Prim). **SCC (Tarjan / Kosaraju), 2-SAT**. Bridges / articulations. Euler paths. **Max-flow** (Dinic, push-relabel) / **min-cut / min-cost flow**. Bipartite matching (Hopcroft-Karp). Strings: KMP, Z-function, suffix array / automaton, **Aho-Corasick**, rolling hash. **Game theory** (Sprague-Grundy / Nim). Geometry: convex hull, line sweep, half-plane. **Number theory**: mod inverse, CRT, factorization, Miller-Rabin, Pollard-Rho. **FFT / NTT / polynomial**. Matrix exponentiation. Berlekamp-Massey. Prob + expected value. Catalan / Stirling / partitions.

## Contests
**Codeforces** (Div 1/2/3/4). **AtCoder** (ABC / ARC / AGC). **ICPC** (regionals → world finals). **IOI**. Google Code Jam + Kick Start (archived). Meta Hacker Cup. Topcoder SRMs. LeetCode contests. CS Academy. HackerRank, HackerEarth.

## Ladder / levels (Codeforces rating approx)
- Newbie → Pupil → Specialist (< 1600)
- Expert (1600-1900), Candidate Master (1900-2100)
- Master / IM (2100-2400)
- GM / LGM (2400+)

Most FAANG interview problems topped at Specialist / Expert level.
